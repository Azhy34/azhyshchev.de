Architecture

How it's built

  • Two-model lyrics pipeline: Gemini 2.5 Flash generates structure and rhyme candidates, Claude Sonnet 4.6 refines for quality and emotional impact.
  • Suno V5.5 API for music generation — selected after testing 4 different music APIs.
  • Shopify for payments — embedded checkout without custom payment infrastructure.
  • Magic link auth — no passwords, frictionless for non-technical users.
  • Personal cabinet: order history, track downloads, status tracking.
Product decisions

Key choices

  • Tested Suno, Udio, and two others — chose Suno V5.5 for consistent output quality at scale.
  • Two LLMs for lyrics instead of one — Gemini is faster and cheaper for drafts, Claude is better at emotional nuance for final output.
  • Shopify over custom payments — 2 weeks is not enough time to build payment infrastructure properly.
  • VPS over serverless — predictable costs for a music generation workload that has long API response times.
