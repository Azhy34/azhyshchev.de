AI Image Composer

A Polish creative studio was spending hours on trial-and-error prompting to produce personalized AI paintings for clients. The bottleneck was identity replacement — getting the client's face into a scene without quality loss. Built a production web app in one day that turns this into a 30-second workflow.

30 secPer artwork from upload to result — was hours of manual work per image

Mobile-firstOptimized for phones and field usage, not just desktop demo
How it works

The workflow

  • User uploads source photo + selects style template from the UI.
  • Canvas API extracts face region with masking — no external preprocessing.
  • Gemini multimodal API performs identity-preserving style transfer in the cloud.
  • Result delivered in the browser within seconds, ready to download or share.
Engineering choices

Why this approach

  • Chose AI generation over geometric transformation (SAM2 segmentation) — adoption mattered more than technical elegance.
  • React 19 for instant UI feedback without page reloads.
  • Railway deployment for zero-config backend scaling.
  • Mobile-first design: target users work on phones at client visits.
Business impact

What changed

  • Hours of manual prompting per artwork → 30 seconds per artwork.
  • 85-90% daily team adoption from day one — because it was simpler than the old workflow.
  • Studio can now handle 10x more client orders without additional staff.
  • Same-day delivery of personalized artwork replaced next-day or longer turnaround.
